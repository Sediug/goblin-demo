# Goblin DB Demo
Demo for Goblin DB v1 features

# Running demos

1. Clone this repository
```bash
    git clone https://github.com/UlisesGascon/GoblinDB 
```

2. Enter in the folder
```bash
    cd GoblinDB
```

3. Install the dependencies
```bash
    npm install
```

4. Run *demos* and see the result.
```bash
    npm run set
    npm run push
    npm run update
    npm run delete
    npm run truncate
```
